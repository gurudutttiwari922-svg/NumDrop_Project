# NumDrop

## Overview
NumDrop is a grid-based number merging game built in Java.  
It features gravity mechanics, chain reactions, and advanced merging logic inspired by puzzle games.

## Core Mechanics
- Vertical merging of blocks
- Horizontal suction-based merging
- Gravity system
- Chain reactions
- Dynamic spawning based on board state

## Architecture
The project follows a modular layered design:

- **core** → Game logic (merge, gravity, chain)
- **model** → Data structures (Board, Block, GameState)
- **input** → User input handling
- **view** → Console rendering
- **main** → Entry point
